# TinyGpt
This repository contains an implementation of TinyGPT, a small-scale GPT model created from scratch

## Configuration

| Setting          | Value |
| ---------------- | ----- |
| attn_dropout     | 0.1   |
| embed_dropout    | 0.1   |
| ff_dropout       | 0.1   |
| vocab_size       | 100   |
| max_len          | 20    |
| num_heads        | 12    |
| embed_dim        | 768   |
| num_decoder_blocks | 12   |
